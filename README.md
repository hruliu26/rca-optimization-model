# RCA Optimization Model

A mechanistic model to predict and optimize RCA yield based on incubation time, template concentration, polymerase concentration, primer concentration, dNTP concentration, and magnesium concentration.

## Pre-requisites
- **Python 3.9+**

## Installation
1. Clone the repository
```bash
git clone [your-repo-url]
cd rca-optimization-model
```

2. Create + activate virtual environment
```bash
python -m venv .venv
source .venv/bin/activate # Windows: .venv/Scripts/activate
```

3. Install dependencies
```bash
python -m pip install -r requirements.txt
```

## Running the model
```bash
python main.py
```
